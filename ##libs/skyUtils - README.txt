[Instructions]
skyUtils funtions:

 - void haltf():
Halt the process execution until a button is pressed.

 - void inlineClear(int x):
Clear x characters in the terminal starting from the outermost right character in the current row towards left.
Works for the current row only.

 - void sep(int width, int tab):
Prints a separation like ( ======== ).
Width regulates the length and tab the indentation, in characters.

 - void dieStruct(int x):
Prints a die face, with x as value. x should range within 1 to 6, else you may occur in bugs.

 - void screenClear(): [Windows only]
Clears all the terminal.

[Istruzioni]
Funzioni skyUtils:

 - void haltf():
Interrompe l'esecuzione del processo finché un pulsante non è premuto.

 - void inlineClear(int x):
Cancella x caratteri nel terminale dal carattere più esterno destro verso sinistra.
Funziona solo per la riga corrente.

 - void sep(int width, int tab):
Stampa un separè come ( ======== ).
Width regola la lunghezza e tab l'indentazione, in caratteri.

 - void dieStruct(int x):
Stampa la faccia di un dado, con x come valore. x dovrebbe variare da 1 a 6, o si può o correre in bug.

 - void screenClear(): [solo Windows] 
Pulisce tutto il terminale.
